1. Bubble Sort the list: 33, 56, 17, 8, 95, 22。Make sure the final result is
from small to large.
Write out the list after the 2nd pass. (10 points)
```
int main(){
    int data[6];
    for(int i = 0; i < 6; ++i){
        scanf("%d", &data[i]);
    }
    for(int i = 0; i < 5; ++i){//i-限制最后一位为最大值，不参与排序 
		for(int j = 0; j < (5 - i); ++j){//j-真实的操作量 
			if (data[j] > data[j+1]){
			    int temp; 
		        temp = data[j];
			    data[j] = data[j+1];
		        data[j+1] = temp;
		    }
		}
	}
}

```
